# Simple Linear Regression in Machine Learning

This repository demonstrates the implementation of Simple Linear Regression as a fundamental machine learning algorithm. It explains how a linear relationship between input (independent variable) and output (dependent variable) is used to make predictions on continuous data.

## Topics Covered

- Introduction to regression in machine learning
- Understanding linear relationship between variables
- Hypothesis function (linear equation)
- Cost function (Mean Squared Error)
- Model training and fitting
- Prediction on new data
- Model evaluation techniques (R² score, error metrics)
- Visualization of regression line
